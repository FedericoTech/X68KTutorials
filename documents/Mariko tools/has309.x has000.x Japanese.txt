使用法：as ［スイッチ］ ファイル名
        -t path         テンポラリパス指定
        -o name         オブジェクトファイル名
        -i path         インクルードパス指定
        -p [file]       リストファイル作成
        -n              最適化の禁止
        -w [n]          ワーニングレベルの指定(0～4)
        -u              未定義シンボルを外部参照にする
        -d              すべてのシンボルを外部定義にする
        -8              シンボルの識別長を８バイトにする
        -s symbol[=num] シンボルの定義
        -x [file]       シンボルの出力
        -f [f,m,w,p]    リストファイルのフォーマット指定
        -l              起動時にタイトルを表示する
        -e              外部参照オフセットのデフォルトをロングワードにする
        -b              ロングワードのPC間接を絶対ロングにする
        -g              SCD用デバッグ情報の出力
        -c              HAS v2.x互換の最適化を行う
        -m 680x0        アセンブル対象CPUの指定

        環境変数 HAS の内容がコマンドラインの最後に追加されます
