使用法：make ［スイッチ｜マクロ定義｜ゴールターゲット名］・・・
        /a      タイムスタンプを無視して考査する
        /c path ディレクトリを変更する
        /d      デバッグ情報を表示する
        /D      更新考査を終えるたびにデータベースを表示する
        /e      環境変数がファイルでの定義を覆うようにする
        /f file 'file'をメイクファイルとして読み込む
        /h      このメッセージを表示する（他には何もしない）
        /i      コマンドのエラーを無視する
        /I path 'path'をインクルードパスのリストに加える
        /k      エラーが起こっても他の考査を続ける
        /n      更新を行わない
        /N      考査を行わない
        /o name 'name'を考査しない
        /p      データベースを表示する
        /q      更新はせず、考査結果を終了コードとして返す
        /r      内蔵規則を使用しない
        /R name 'name'を必ず更新する
        /s      コマンド行を表示しない
        /S      先に指定された/kスイッチを取り消す
        /t      タイムスタンプのみを更新する
        /v      バージョン情報を表示する
        /w      カレントディレクトリを表示する
        /W name 'name'を真新しいものとみなす
