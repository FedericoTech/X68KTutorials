Usage: lk [switch] file [file...]
         /m nn maximum number of symbols (201<nn<65536)
         /t path Specify temporary path
         /o file object file name
         /b address base addressing
         /i file specify indirect file
         /v verbose mode
         /x disable output of symbol table
         /p output map file
         /a n X_File option bit specification (0-7)
         /e nn Object alignment setting (2,4,8,16...32768)
